# College Course Registration System

A simple LLD (Low-Level Design) project in C++ to practice OOP by modeling a college course registration system.

## What it does

- Add departments to the college
- Add courses and professors to a department
- Assign courses to professors, avoiding slot clashes
- Add students to the college
- Register students for courses, avoiding slot clashes

## Tech Stack

- C++
- OOP (classes, encapsulation, composition)

## Ideas to Extend

- Add prerequisites for courses
- Add a max capacity per course/section
- Allow students to drop/swap courses
- Generate a timetable view per student or professor
- Add grades and a GPA calculator
- Persist data to a file or simple database
